# NAME

citeproc - process citations using a CSL stylesheet.

# SYNOPSIS

citeproc [options] [file]

# DESCRIPTION



# OPTIONS

`-r` *FILE*, `--references`=*FILE*
:   Specify a CSL JSON bibliography to be used as a source for
    references.

`-h, --help`
:   Print usage information.

`-V, --version`
:   Print version.
